# lp-fitlife
Landing Page para Fit Life, app para consultoria esportiva que ajuda as pessoas a manterem uma vida saudável com planos de treino e nutrição personalizados.

<b>Estrutura da Landing Page</b>
- Header
Logo do aplicativo
Menu de navegação com links para diferentes seções (Home, Features, Pricing, Contact)

- Hero Section
Uma imagem de fundo atrativa
Título e subtítulo chamativos
Um botão de chamada para ação (CTA) como "Get Started" ou "Sign Up Now"

- Features Section
Três ou quatro colunas destacando as principais funcionalidades do aplicativo
Cada coluna deve ter um ícone, um título e uma breve descrição

- Pricing Section
Três planos de preços (Free, Basic, Pro)
Cada plano deve ter um título, preço, lista de benefícios e um botão de CTA

- Testimonials Section
Depoimentos de usuários satisfeitos
Cada depoimento deve ter uma foto do usuário, nome e o texto do depoimento

- Contact Section
Um formulário de contato simples com campos para nome, email e mensagem
Botão de envio

- Footer
Links para redes sociais
Informações de contato
Direitos autorais
